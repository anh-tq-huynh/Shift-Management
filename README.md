# Shift Management Backend 
<b>Note: README file is in progress. </b>
- A Spring Boot backend for managing companies, employees, clients, and shift‑related operations.  

---

## 🚀 Features

### **Company Management**
- Create companies  
- Retrieve all companies  
- Retrieve company details  
- Assign employees to a company  
- Assign clients to a company  

### **Employee Management**
- Create and list employees  
- Hire employees into a company  

### **Client Management**
- Create and list clients  
- Add clients to a company  

---

## 🛠️ Tech Stack

- **Java 17**
- **Spring Boot**
- **Spring Web**
- **Spring Data JPA**
- **Hibernate**
- **H2 / PostgreSQL** (depending on your configuration)
- **Maven**

---

## 📁 Project Structure


- **Controller layer** exposes REST endpoints  
- **Service layer** contains business logic  
- **Repository layer** handles database operations  
- **Model layer** defines entities and relationships
